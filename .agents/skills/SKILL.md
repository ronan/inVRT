---
name: todo-start-next
description: How to start the next item on the todo list
---

The next task to be planned is the first item on the todo list. 

1. Use the instructions in AGENTS.md to find the next item on the todo list
2. Create a plan to accomplish the todo. Use the instructions attached to the todo item
3. Add an item to the plan to move the todo item to `docs/planning/TODO-DONE.md` when done
4. Save the plan to the `docs/planning/agent-plans/` directory.
5. Do not implement the plan. This is just the planning stage.
