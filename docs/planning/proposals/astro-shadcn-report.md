# Build a single-page Playwright Test Report generator using Astro and Shadcn/UI.

# Core Requirements:

## Framework: 

Use Astro with the Tailwind CSS integration.
    - UI Components: Use Shadcn/UI for the interface
    - Use the shadcn preset b7C9rv8KW (npx shadcn@latest init --preset b7C9rv8KW --template astro)

## Data Source: 
    - The page should pull data from plan.yml and the playright results.

## The Layout: 

Create a dashboard header with "Stats Cards" (Total, Unchanged Pages, Changed Pages) followed by a searchable/filterable list of pages.

Each page in the list should show the page title, page path, whether it has changes and other useful test info

Each page should have a thumbnail generated from the reference bitmap for the page.

## Single File Output:

Configure the project to bundle all CSS and JS into a single standalone HTML file (using astro-single-file or a similar method) so the report can be shared via Slack or email.

## Integrate with the application

Update the playwright configuration generated by the configure command to trigger the astro build.
